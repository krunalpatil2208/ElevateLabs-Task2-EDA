# Task 2 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset to understand its structure, patterns, and relationships using statistics and visualizations.

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Dataset

Titanic Dataset

---

## EDA Steps

- Loaded the dataset
- Explored dataset information
- Generated summary statistics
- Checked missing values
- Created histograms
- Created boxplots
- Generated correlation matrix
- Created pairplot
- Visualized passenger class and survival
- Derived insights from visualizations

---

## Visualizations

- Histogram
- Boxplot
- Correlation Heatmap
- Pairplot
- Countplots

---

## Key Insights

- Majority of passengers were between 20–40 years old.
- Fare contained several outliers.
- Most passengers belonged to 3rd class.
- Female passengers had a higher survival rate.
- First-class passengers survived more frequently.
